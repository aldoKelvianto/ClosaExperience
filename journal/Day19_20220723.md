# App lifecycle and Fastlane mapping

| Step no. | Manual step  | Fastlane                   |
|----------|--------------|----------------------------|
| 1        | Apple ID     |                            |
| 2        | Registration | Produce                    |
| 3        | Team setup   | Cert, Sign, Match          |
| 4        | Capabilities |                            |
| 5        | Testing      | Scan, Gym, Pilot           |
| 6        | Distribution | Snapshot, Deliver, Frameit |
| 7        | App sales    |                            |
- Fastlane is a bunch of Ruby files for common automation. 
- Fastfile is a Ruby DSL that will be parsed by fastlane program
- Install Fastlane by using brew, `brew install fastlane`
