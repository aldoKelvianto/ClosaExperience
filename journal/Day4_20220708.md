- Just like Java, there is no built-in support for singleton in Swift. In Kotlin, we have `object`. To create a singleton, you can use:

1. global variable
2. static on class or struct

This begs a question, which one we should use? Use class for mutable singleton and struct for immutable singleton. Although you can make struct as an mutable singleton by declaring your field as `var` instead of `let`. But as a convention, struct is for immutable singleton.

```swift
struct PersistenceController {
	static let shared = PersistenceController()
}

struct WeatheringWithSwiftApp: App {
	let controller = PersistenceController.shared
}
```
